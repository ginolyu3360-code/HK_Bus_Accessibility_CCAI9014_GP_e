# HK-Bus-Accessibility-Analysis
A network-based pedestrian accessibility model for Hong Kong's bus transit system.

## Overview
This project evaluates the real-world accessibility of bus stops in Hong Kong using **OpenStreetMap (OSM)** street networks. Unlike simple buffer analysis, this model calculates the 400m walking distance along actual road topologies.

## Key Features
- **Multi-Agency Integration**: Real-time stop data from KMB, Citybus, and NLB.
- **Network Topology**: Uses Dijkstra's algorithm via `NetworkX` for accurate service area modeling.
- **Regional Analysis**: Segmented processing for Kowloon, HK Island, and New Territories.
